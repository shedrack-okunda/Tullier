## ğŸš€ Getting Started

### Prerequisites

-   Node.js 16+
-   npm or yarn package manager

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/tullier-wellness.git
cd tullier-wellness
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Start the development server:

```bash
npm run dev
# or

```

```bash
npm run build
# or
```
